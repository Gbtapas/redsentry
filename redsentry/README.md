# RedSentry — Red Team Automation Tool

Ferramenta de automação para reconhecimento e análise de segurança, desenvolvida em Python.

## Funcionalidades

- **Port Scanner** — Scan de portas TCP comuns com detecção de serviços
- **CVE Search** — Busca automática de vulnerabilidades conhecidas via API do NVD
- **Subdomain Enumeration** — Enumeração de subdominios por wordlist
- **DNS Collector** — Coleta de informações DNS e WHOIS do alvo
- **Report Generator** — Geração de relatório HTML com todos os achados e classificação de risco
